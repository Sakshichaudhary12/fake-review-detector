# fake-review-detector

This is a machine learning project that detects whether a product review is Fake or Genuine.  
It uses TF-IDF features and a Naive Bayes classifier trained on a large labeled dataset.

---

## Features
- Clean Streamlit Web UI
- NLP preprocessing (tokenizing, stopwords removal, stemming)
- TF-IDF vectorization
- Trained ML model for prediction
- Easy to deploy on Streamlit Cloud
